两数之和

题号：1
描述：
给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 的那 两个 整数，并返回它们的数组下标。你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。你可以按任意顺序返回答案。
示例1：
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

题解一：

​ 首先想到的肯定是暴力破解法，通过题目的描述，我们仅需要将数组中的数字两两组合（不需要重复组合），并且判断和是否为target即可。

题解二：

​ 题目中，我们每得到一个数字，就需要与数组中数字进行判断和是否为target。我们可以借助哈希表，实现快速取数并进行比对。

  1. 遍历nums数组
  2. 在哈希表中查找，是否有target-nums[i]的值，有则返回i和哈希表target-nums[i]的值，没有则存入nums[i]和下标i。

整数反转

题号：7

描述：给你一个 32 位的有符号整数 x ，返回 x 中每位上的数字反转后的结果。

如果反转后整数超过 32 位的有符号整数的范围 [−2^31^, 2^31^ − 1] ，就返回 0。

假设环境不允许存储 64 位整数（有符号或无符号）。

示例1：

输入：x = 123
输出：321

题解一：

​ 提取出每一位的数字，并重新组合。

  1. 将x%10得到个位值，并加上result*10
  2. 将x / 10舍去个位数字，并将十位数字变为各位数字，同步骤1操作。
  3. 当x为0时，循环结束
  4. 判断结果是否溢出，在循环中，result只要大于int类型最大值/10,或者小于int类型最小值/10都会溢出。

题解二：

​ 与题解一思路相同，这里提供另外一种判断溢出的方法。int类型占4字节会有溢出，将结果给一个long类型的变量（占8字节），最终结果将long类型强制转换为int类型并且与原long类型比较大小，若相同则没有溢出，若不相同则溢出。
