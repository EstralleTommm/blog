  • 队列初始化函数kfifo_init需要放在函数体内！！！

  • sizeof返回的是一个数据类型占用的内存大小（字节）；

    strlen是从内存的某个位置（可以是字符串开头，中间某个位置，甚至是某个不确定的内存区域）开始扫描，直到碰到第一个字符串结束符’\0’为止，然后返回计数器值(长度不包含’\0’) ；

  • 函数说明

    kfifo_from_user (struct __kfifo fifo, const void from, unsigned int len, unsigned int *copied)

    • *fifo : 要使用的fifo的地址 ;
    • *from: 数据来源 （用户空间指针） ;
    • len:数据长度;
    • copied：拷贝成功的字节数 ;

    kfifo_to_user (struct __kfifo fifo, const void from, unsigned int len, unsigned int *copied)；

    • *fifo : 要使用的fifo的地址 ;
    • *from :目标地址（内核空间的指针）
    • len:数据长度;
    • copied：拷贝成功的字节数 ;
